# Formata Fácil

## Sobre o Projeto
O **Formata Fácil** é um site responsivo criado para divulgar serviços de formatação de notebooks e desktops, backup de dados e instalação de programas. Com um design moderno e amigável, o site oferece informações claras sobre os serviços e permite que os usuários entrem em contato facilmente.

### Demonstração
Acesse o site publicado no GitHub Pages:
**[Formata Fácil](https://seu-usuario.github.io/FormataFacil/)**

---

## Funcionalidades
- Apresentação clara dos serviços oferecidos.
- Formulário de contato para solicitações e dúvidas.
- Design responsivo para navegação em diferentes dispositivos.

### Estrutura do Projeto
1. **Navbar**: Contém o logotipo e links para as seções principais.
2. **Hero Section**: Destaque visual com chamada principal.
3. **Serviços**: Descrição de cada serviço com ícones explicativos.
4. **Contato**: Formulário para envio de mensagens.
5. **Rodapé**: Direitos autorais e informações complementares.
